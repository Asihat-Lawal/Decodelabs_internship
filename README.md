# E-Commerce Revenue Analysis

## Project Overview

The objective of this project was to clean and analyze e-commerce transaction data, uncover sales trends, and generate actionable business insights through exploratory data analysis (EDA).

## Skills Demonstrated

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Revenue Analysis
* Business Insight Generation
* Microsoft Excel
  
## Key Metrics Analyzed

- Total Revenue
- Average Order Value (AOV)
- Revenue by Product
- Revenue by Year
- Order Status Distribution
- Payment Method Performance

## Dataset Information

The dataset contains:

* 1,200 transaction records
* Customer information
* Product categories
* Revenue data
* Payment methods
* Order statuses
* Referral sources

## Analysis Performed

### Revenue Statistics

* Total Revenue by Year
* Average Order Value (AOV)

### Exploratory Data Analysis

* Revenue trends
* Product performance analysis
* Order status analysis
* Payment method analysis

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Descriptive Statistics

## Key Outcome

The project provided insights into revenue performance and customer purchasing behavior while demonstrating practical data cleaning and analytical skills.
